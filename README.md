课程介绍：

有应用的地方就会有搜索，有搜索的地方就能使用 Elasticsearch 来实现，Elasticsearch 用起来简单，但想要做好还是有很高的技术门槛的。要用 Go 语言构建搜索服务，并完成海量数据的优化方案，缺乏经验就会有诸多问题暴露，难免走弯路。本课程通过对基础技能到实战项目的深度解读，带你夯实 ES+Go 的底层技能，积累实战经验，一举跃升经验丰富的 Go 语言开发者。
